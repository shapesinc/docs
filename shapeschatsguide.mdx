    ## Chats – Your Social Hub

    Chats are where the fun unfolds — chat, collaborate, or roleplay with Shapes and friends.

    ### Create a Chat

    1. Open your side navigation menu.
       - **Web:** visible by default
       - **App:** tap the ☰ menu icon (top left)
    2. Select **New Chat**.
    3. Add Shapes, people, or usernames.
    4. On the details step, confirm the chat name and any advanced settings you want.
    5. Create the chat and start talking.

    [Detailed Chat Creation Guide](/create-a-chat)

    ![New Chat Example](/images/newroom.png)

    ---

    ### Adding Shapes

    1. Click **Chat Banner → Add Shapes**
    2. Choose one:
       - Search for new Shapes
       - Invite recently used Shapes
       - Type a Shape's username

    ![Chat Banner Example](/images/roombanner.png)

    ---

    ### Inviting Friends

    - Click **Chat Banner → Invite Friends**
    - Share a link on social media
    - Copy-paste the invite
    - Add directly from Shapes contacts

    ---

    ### Activating Shapes

    - Click **Chat Banner → Participants**
    - Tap ⋯ next to a Shape → **Activate**
    - Or type `/activate @shape-name`

    Deactivate anytime:\
    `/deactivate @shape-name`

    ---

    ### Chat Aesthetics

    Give your chat personality!\
    Change the banner, background, fonts, and colors to match your vibe.

    [Learn more about Chat Styling](/chat-appearance)

    ---

    ### Skip Turn and Fast Forward

    Want quick replies?\
    Open the **+** menu beside the message box and choose **Skip turn and fast forward**.

    - Pick the next Shape that should reply
    - Choose 5 or 10 messages to let Shapes continue automatically
    - Stop fast forward while it is running

    ---

    ## Quick Commands & Tricks

    | Command              | What It Does                                      |
    | -------------------- | ------------------------------------------------- |
    | `/activate @shape`   | Activates a Shape                                 |
    | `/deactivate @shape` | Deactivates a Shape                               |
    | `/wack`              | Clears short-term memory for the current Shape    |
    | `/wack-all`          | Clears short-term memory for Shapes in the chat   |
    | `/reset`             | Resets the chat context                           |
    | `/sleep`             | Puts a Shape to sleep                             |
    | `/imagine`           | Generates an image                                |
    | `/web`               | Asks a Shape to use web search                    |
    | `/poll`              | Creates a poll in supported chats                 |
    | `/games`             | Opens chat games when the chat supports them      |
    | `Pin messages`       | Keeps key info at the top                         |
    | `Skip turn and fast forward` | Lets Shapes continue the conversation     |

    Here you can customize your Chat with options such as:

    - Bubble colors

    - Chat description
    - Chat tags

    For more information about Chat Appearance, read the [Chat Appearance Guide](/chat-appearance).

      ## Shape Free Will

      Shape Free Will is controlled per chat from **Chat Settings → AI Configurations**. It decides when Shapes can jump into that chat on their own.

      | Setting                           | Description                                      |
      | --------------------------------- | ------------------------------------------------ |
      | **Reply when mentioned**          | Shapes respond when someone uses their @name.    |
      | **Keep the convo going**          | Shapes may continue after they have replied.     |
      | **React to keywords**             | Shapes can jump in when chat keywords appear.    |
      | **Always have something to say**  | Shapes can reply when nobody else does.          |
      | **Come back later**               | Shapes may check back in after the chat is quiet. |

      <Accordion title="Example Behavior Styles">
        Turn on more triggers for an outgoing chat. Keep only mentions enabled if you want Shapes to wait until called.
      </Accordion>

    # AI Engine

    Your Shape’s engine model - configure how it thinks, creates, and responds.\
    This is one of the most important tabs on your Shape’s dashboard.

    On this tab, you can:

    - Update your Shape’s **Primary model** and **Fallback model**
    - Enable or disable **internal monologging**
    - Toggle helpful options:
      - Inform User When Fallback Engine is Used
      - Inform Users When They Are Out of Shape Engine Credits
      - Let Users Override AI Engine
    - Select your Shape’s general **Language Preset**
    - Write a **Base Preset** that defines its tone and behavior

      > Presets heavily impact how your Shape expresses personality.

    | Setting             | Description                              |
    | ------------------- | ---------------------------------------- |
    | **Primary Engine**  | Premium model used by default            |
    | **Fallback Engine** | Free backup for non-premium users        |
    | **Language Preset** | Defines your Shape’s base language       |
    | **Base Preset**     | Core writing style and tone              |
    | **Monologue**       | Enable/disable internal thought messages |

    ---

    ### Advanced Engine Parameters

    <Accordion title="Fine-Tune Engine Controls">
      You can also edit advanced AI parameters for more granular control:

      - **Temperature** — randomness of responses
      - **Top P** — controls probability diversity
      - **Repetition Penalty** — reduces repeated text
      - **Top K** — limits token sampling
      - **Frequency Penalty** — penalizes overused words
      - **Presence Penalty** — encourages new ideas
      - **Min P** — minimum probability cutoff
      - **Top A** — adaptive sampling
      - **Timezone** — define the time context your Shape uses

      > ⚠️ It’s not recommended to edit these settings unless you know what you’re doing — small changes can drastically affect how your Shape behaves.
    </Accordion>
    ## Memory Systems

    Shapes remember what matters.

    | Type                        | Description                                    |
    | --------------------------- | ---------------------------------------------- |
    | **Short-Term Memory (STM)** | Keeps recent messages for context              |
    | **Long-Term Memory (LTM)**  | Stores persistent facts and relationships      |
    | **Memory Recall**           | Retrieves memories based on relevance          |
    | **Relevance**               | Controls how selective memory recall should be |

    <Accordion title="Memory Tips">
      - Lower relevance = more recall, less accuracy
      - Higher relevance = fewer recalls, more precision
      - Use STM for recent context and LTM for lore or history
    </Accordion>

    ### Change Your Display Name or Username

    - Click the ✏️ icon beside your display name or username
    - Type your new name
    - Click the **X** in the top-right corner to save

    <Warning>
      Don’t press “Cancel” — your changes save automatically.
    </Warning>
    ![Display Name Editing Example](/images/displayname.png)

    ### Profile Picture & Banner

    - Click the ✏️ icon beside your **profile picture** or **banner**
    - Choose to:
      - Upload an image (max **4MB**)
      - Remove the current one
      - Generate an **AI banner**
    - Perfect for showcasing your vibe!

    ### Bio / Description

    - Click the ✏️ icon beside "_Add a bio to tell others about yourself..._"
    - Type your new bio
    - Click the **X** to save

      - Turning off **User Level** hides level icons next to your name
