## Overview

The Shapes API is built on OpenAI-compatible standards, making integration straightforward with existing libraries and SDKs. Each Shape comes with:

- **Rich Personalities**: Fully developed character traits and behaviors
- **Memory Systems**: Both short-term and long-term memory across platforms
- **Multimodal Support**: Text, image, and voice capabilities
- **Tool Integration**: Support for custom tools and MCP functionality

## Getting Started

### 1. Generate API Key

First, obtain your API key from the [developer dashboard](https://shapes.inc/developer).

### 2. Base Configuration

| Parameter | Value |
|-----------|-------|
| Base URL | `https://api.shapes.inc/v1/` |
| Model Format | `shapesinc/<shape-username>` |
| Authentication | Bearer token in Authorization header |

### 3. Environment Variables

Set up your environment variables for easier integration:

```bash
export SHAPESINC_API_KEY="your-api-key-here"
export SHAPESINC_SHAPE_USERNAME="your-shape-username"
```

## Authentication

### API Key Authentication

Include your API key in the Authorization header:

```
Authorization: Bearer <your-API-key>
```

### Custom Headers

For production applications, we strongly recommend using custom headers:

| Header | Purpose | Required |
|--------|---------|----------|
| `X-User-Id` | User identification | Recommended |
| `X-Channel-Id` | Conversation context | Optional |

<Warning>
Without custom headers, all API requests are treated as coming from the API key owner, which can lead to conversation context mix-ups and memory issues.
</Warning>

## API Specifications

### Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/chat/completions` | POST | Send messages to a Shape and receive responses |

### Rate Limits

- **Default**: 20 requests per minute
- **Enterprise**: [Contact us](https://docs.google.com/forms/d/e/1FAIpQLScGLeRk6snViRPslXbbUaMDwubcBhmcJ6opq7wFvPEp-EbO3g/viewform) for higher limits

### Response Format

All responses follow the OpenAI-compatible JSON format:

```json
{
  "id": "chatcmpl-123",
  "object": "chat.completion",
  "created": 1677652288,
  "model": "shapesinc/your-shape",
  "choices": [{
    "index": 0,
    "message": {
      "role": "assistant",
      "content": "Hello! I'm doing great, thank you for asking."
    },
    "finish_reason": "stop"
  }],
  "usage": {
    "prompt_tokens": 9,
    "completion_tokens": 12,
    "total_tokens": 21
  }
}
```

## Multimodal Support

### Image Processing

Send images alongside text messages:

```json
{
  "model": "shapesinc/your-shape",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "What do you see in this image?"
        },
        {
          "type": "image_url",
          "image_url": {
            "url": "https://example.com/image.jpg"
          }
        }
      ]
    }
  ]
}
```

### Audio Processing

Process audio files with automatic transcription:

```json
{
  "model": "shapesinc/your-shape",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "Please respond to this audio message"
        },
        {
          "type": "audio_url",
          "audio_url": {
            "url": "https://example.com/audio.mp3"
          }
        }
      ]
    }
  ]
}
```

**Supported formats**: MP3, WAV, OGG

<Note>
Currently, only one media file (image or audio) can be processed per message.
</Note>

## Shape Commands

Shapes respond to special commands for enhanced functionality:

| Command | Description |
|---------|-------------|
| `!reset` | Clear the Shape's long-term memory |
| `!sleep` | Generate long-term memory snapshot |
| `!dashboard` | Access Shape configuration |
| `!info` | Get Shape information |
| `!web` | Search the web (free for all users) |
| `!help` | Display available commands |
| `!imagine` | Generate images |
| `!wack` | Clear short-term memory |

## Current Limitations

<Warning>
Be aware of these current API limitations:
</Warning>

- **No System Messages**: Shape personality is configured through the Shapes.inc dashboard
- **No Message History**: API relies on Shape's built-in memory systems
- **No Streaming**: Only complete responses are supported
- **Fixed Parameters**: Temperature and other settings are controlled via Shape configuration
- **Single Media**: Only one image or audio file per message

## Error Handling

Common error responses and their meanings:

| Status Code | Error | Description |
|-------------|-------|-------------|
| 401 | Unauthorized | Invalid or missing API key |
| 404 | Not Found | Shape username not found |
| 429 | Rate Limited | Too many requests |
| 500 | Server Error | Internal server error |
