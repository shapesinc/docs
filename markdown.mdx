### Quick Reference Table

| Markdown                      | Output                |
| :---------------------------- | :-------------------- |
| `**bold**`                    | **bold**              |
| `*italic*`                    | _italic_              |
| `***bold and italic***`       | **_bold and italic_** |
| `~~strike~~`                  | ~~strike~~            |
| `# Header `                   | # Header              |
| `## Header `                  | ## Header             |
| `### Header `                 | ### Header            |
| `> quote`                     | \> quote              |
| `inline code`                 | `inline code`         |
| \`\`\`code\`\`\`              | `code block`          |
| `- item`                      | - item                |
| `1. item`                     | 1. item               |
| `[link](https://example.com)` | link                  |
