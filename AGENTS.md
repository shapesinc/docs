# Documentation terminology

- The platform, product, and company name is **Shapes, Inc.** Always include the comma and period.
- Never refer to the platform as **Shapes**. For example, write **On Shapes, Inc.** instead of **On Shapes**.
- A **Shape** is an AI character on Shapes, Inc. Use **Shapes** only as the plural of those characters.
- Preserve exact interface labels when documenting navigation. Labels such as **Your Shapes** refer to AI characters and are correct.
- Before finishing documentation changes, check that every use of **Shapes** clearly refers to AI characters; otherwise, change it to **Shapes, Inc.**
