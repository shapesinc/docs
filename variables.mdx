#### **Supported Variables**

1. `{shape}` – Refers to the shape itself.
2. `{user}` – Refers to the user interacting with the shape.

You can use these variables across various fields in your shape's dashboard to guide its responses, set presets, and enhance interactions.

### **Where to Use Variables**

#### **1. Profile Tab**

- You can use {shape} and/or {user} in your shape's initial message to make the interaction feel more natural\!
- You can use {shape} variable to define its appearance.
