## How to Create a Chat

This is the simple version. Start a chat first, then add people or Shapes from the chat menu.

### 1. Log in

Open Shapes. If you are already in the app, stay there. If not, open [shapes.inc](https://shapes.inc) or [talk.shapes.inc](https://talk.shapes.inc) and sign in.

### 2. Click New Chat

Open the sidebar and click **New Chat**.

On mobile, open the sidebar first if it is hidden.

### 3. Start the chat

You can start in a few ways:

- Click **Quick ChatGPT** to start fast with a smart Shape that can help you get work done.
- Stay on **Shapes** and pick a Shape from the list.
- Use the search box to find a Shape by name or username.
- Switch to **People** if you want to start with a person instead.

When the chat opens, send a message to get it going. Chats started with **Quick ChatGPT** may get a name from the conversation. For other chats, name the chat yourself so people know what it is for.

### 4. Add people or more Shapes

Open the chat menu:

1. Click **More actions** in the top-right of the chat.
2. Click **Members / Invite**.
3. Choose one of these:

- **Invite friends** to share a link.
- **Add participants** to add people or Shapes to the chat.
- **Create shape** if you want to make a new AI for the chat.

In **Add participants**, use **Shapes** to add AIs or **People** to add users. You can search by name or username.

### 5. Manage the chat

Click **More actions** → **Settings** to open the chat menu.

From there you can find:

- **Pinned Messages**
- **Media & Files**
- **Participants**
- **Search Messages**
- **Notifications**
- **Personal AI Preferences**
- **AI Configurations**
- **Chat Appearance**
- **Moderation**

## Quick Tips

- To mention a Shape in the chat, type `@` in the message box and pick the Shape.
- To invite friends later, go back to **More actions** → **Members / Invite**.
- To change how the chat looks, open **Settings** → **Chat Appearance**.
- To change how the AI behaves in that chat, open **Settings** → **AI Configurations** or use [Custom Chat Instructions](/custominstructions-chats).
