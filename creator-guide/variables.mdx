## What Are Variables?

Variables act as placeholders that help your shape identify itself and the user it interacts with.

### Supported Variables

1. `{shape}` - Refers to the shape itself.
2. `{user}` - Refers to the user interacting with the shape.

You can use these variables across various fields in your shape's dashboard to guide its responses, set presets, and enhance interactions.

### Where to Use Variables

#### 1. Profile Tab

You can use shape and user in your shape's initial message to make the interaction feel more natural. You can use shape variable to define its appearance.

#### 2. Personality Tab

Use variables in the shape's backstory and personality traits for better contextual understanding.

#### 3. FreeWill Tab

Incorporate variables in freewill presets for improved accuracy and effectiveness.

#### 4. Knowledge Tab

You can use variables to explain knowledge entries and commands. And to define relationships and command functionalities more clearly.

#### 5. AI Engine Tab

Use variables when writing presets to improve their effectiveness. Learn more in the Presets guide.

#### 6. Image Engine Tab

You can use variables to define custom image generation rules and styles.

#### 7. Persona Tab

Define the shape's persona effectively using variables.

By utilizing variables in backstories, presets, and predefined messages, you can make your shape more dynamic and engaging.
