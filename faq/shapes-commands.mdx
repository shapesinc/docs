# Shapes Commands

Shapes support a set of commands on shapes.inc that help you control Shape behavior, manage memory, and access special features.

## Available Commands

All commands on shapes.inc use the `/` prefix.

## Command List

Available commands on [shapes.inc](https://shapes.inc):

<AccordionGroup>
  <Accordion title="/help">
    **Description**: Get a list of all available commands
    
    **Usage**: Type and send `/help` in any chat ui to see what commands are available for that specific Shape.
  </Accordion>
  
  <Accordion title="/reset">
    **Description**: Delete all memories of the Shape
    
    **Usage**: Use `/reset` to completely clear the Shape's memory of your conversation history. This gives you a fresh start.
  </Accordion>
  
  <Accordion title="/sleep">
    **Description**: Save the memories about ongoing conversation
    
    **Usage**: Type and send `/sleep` to have the Shape create a summary of your current conversation for future reference.
  </Accordion>
  
  <Accordion title="/imagine">
    **Description**: Generate an image
    
    **Usage**: Use `/imagine [prompt/description]` to generate images. Describe what you want to see in detail.
  </Accordion>
</AccordionGroup>

## Additional Features

<Tip>
  You can also clear short-term memory by clicking the eraser icon next to the input box in the chat interface.
</Tip>

## Tips for Using Commands

<Steps>
  <Step title="Start with /help">
    Always check what commands are available for each specific Shape, as some may have unique commands.
  </Step>
  <Step title="Use memory commands strategically">
    Use `/sleep` at the end of important conversations to preserve context for future sessions.
  </Step>
  <Step title="Reset when needed">
    If a Shape seems confused or off-track, try the erase button or `/reset` to start fresh.
  </Step>
  <Step title="Experiment with image generation">
    Use `/imagine` with detailed descriptions for better image generation results.
  </Step>
</Steps>
