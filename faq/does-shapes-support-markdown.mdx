title: "Does Shapes, Inc. support Markdown?"
description: "See the Markdown basics supported by Shapes, Inc."
---

Yes. Here's a Quick Reference Table:

| Markdown                      | Output                |
| :---------------------------- | :-------------------- |
| `**bold**`                    | **bold**              |
| `*italic*`                    | _italic_              |
| `***bold and italic***`       | **_bold and italic_** |
| `~strike~`                  | ~strike~            |
| `# Heading `                  | # Heading             |
| `## Heading`                  | ## Heading            |
| `### Heading `                | ### Heading           |
| `> quote`                     | \> quote              |
| `inline code`                 | `inline code`         |
| \`\`\`code\`\`\`              | `code block`          |
| `- item`                      | - item                |
| `1. item`                     | 1. item               |
| `[link](https://example.com)` | link                  |
