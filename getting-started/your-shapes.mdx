If you are looking at a shape's dashboard that is owned by you, you're supposed to see something like this:

### Main Profile

Here you are looking at your shape's profile picture, banner (if it has one), its name, and the date it was created.

Below that, you'll see three buttons (if the shape is connected to X; otherwise, two):

- View on X
- Chat Now
- Email Me

You can click the **View on X** button to visit the shape's X profile, the **Chat Now** button to talk with the shape right away, and the **Email Me** button to email the shape.

Below that you'll see a small box containing your shape's dashboard which you can copy and below that you see some shapes similar to the main shape you're looking at.

This page also contains the tags of that shape and some additional details below if the shape creator has added them\!

![Shape Profile Web](/images/shape_profile.webp)

### Sidebar

On the sidebar, you're looking at many tabs, and you might not know what they are used for. That's exactly what we are going to explain here:

![Sidebar Pn](/images/sidebar.png)

## Public Profile

On this tab, you can:

- Update your shape's profile picture and background image
- Set a vanity URL
- Enter basic information such as category and short tagline
- Configure discovery settings: SEO description, tags, and visibility in search results
- Add a featured prompt and conversation style examples
- Showcase screenshots
- Set advanced SEO details like origin, character universe (if any), and character background
- Add custom profile styling code (HTML and CSS for the profile page)

## Personality

On this tab, you can define:

- Your shape's name, personality, and character
- Advanced details such as personality traits, tone, age, birthday, likes, dislikes, story, and appearance

## Settings

Use this tab to:

- Enable certain filters on your shape
- Clone or delete the shape

## In Chat Personality

On this page, you can:

- Update your shape's in-chat personality, including general preset/style, goals, and conversation examples
- Set custom messages such as the initial message, wack message, error message, and sleep message

## Knowledge

On this tab, you can add:

- Additional instructions or knowledge for your shape
- Knowledge types include general knowledge, text commands, relationships, lore, or others

## Training

Here, you can provide training examples to your shape in user-and-shape format, one at a time.

## AI Engine

This is one of the most important tabs on your shape's dashboard. On this tab, you can:

- Update your shape's:
  - Primary model
  - Fallback model
  - Freewill model (currently unavailable)
- Enable/disable internal monologging
- Toggle options like:
  - Inform User When Fallback Engine is Used
  - Inform Users When They Are Out of Shape Engine Credits
  - Let Users Override AI Engine
- Select the general language your shape will use by choosing a **Language Preset**
- Write a **Base Preset** for your shape
  - Presets highly impact your shape's character
  - [Learn more](https://wiki.shapes.inc/new-to-shapes-guide/faqs/readme/presets)

#### Advanced Engine Parameters

You can also edit:

- Temperature
- Top P
- Repetition Penalty
- Top K
- Frequency Penalty
- Presence Penalty
- Min P
- Top A

**\!\! It is not suggested to edit these settings unless you know what you're doing.**

#### Timezone

You can define a timezone for your shape here.

#### Short-Term Memory (STM)

Enable STM and manage its frequency:

- STM is the number of recent messages (user and shape) the shape will take into context.

#### Long-Term Memory

Enable or disable long-term memory and automatic memory generation. You can also:

- Edit generation frequency
- Select a model to generate long-term memories
- Set instructions for the model
- Configure memory recall settings:
  - **Memory Relevance**:
    - Lower values (0.1) recall more memories but may be less relevant
    - Higher values (1.0) recall fewer but more relevant memories
- Configure:
  - Memory Sharing
  - Memory Privacy
    - **Private mode**: memories aren't shared at any point
    - **Global mode**: memories are shared across all contexts

#### Training Settings

- Jump to the training tab using a button
- Adjust **Training Context Size**: number of relevant training examples to include in conversations
- Adjust **Training Relevance**:
  - Lower values recall more examples but may be less relevant
  - Higher values recall fewer but more matched examples

#### Knowledge Settings

- Jump to the knowledge tab using a button
- Adjust **Knowledge Context Size**: number of relevant knowledge entries to include in conversations
- Adjust **Knowledge Relevance**:
  - Lower values recall more entries but may be less relevant
  - Higher values recall fewer but more relevant entries

## Image Engine

On this tab, you can:

- Set up an **Image Generation Prefix** for your shape
  - If you send a message with this prefix, the shape will generate an image from the text
- Select which **image model** to use for generation
  - Premium engines will cost credits
- Add an **Image Preset**
  - The shape will consider this info while generating images
- Specify the size of generated images
  - Square images are faster to generate
- Enable a toggle to force the shape to always generate images in the selected ratio

## Voice Engine

On this tab, you can:

- Enable/disable voice messages for your shape
- Choose a sample voice or upload your own voice sample for cloning
- Manage how often your shape sends voice messages
- Choose the core voice generation model
- Enable/disable voice transcripts
- Configure advanced voice settings (default values recommended)
- Update or modify the voice preset for your shape

## Persona

On this tab, you can define your **User Persona**:

- Add a name for yourself
- Write a backstory about your character

## Memory

View, delete, and write memories of your shape with you.

## Mail

This is your shape's email address. Clicking on it will:

- Copy the email address
- Open your default email app (if available)
