Hello there, There is a chance you might find our dashboard somewhat complex to explore, so we are writing this guide to help you easily navigate our platform.

## [Homepage](https://shapes.inc/explore)

![Homepage Pn](/images/homepage.png)

Our homepage contains a search bar, a create button, direct tag searches, a list of shapes you own, shapes you've recently interacted with, some other popular shapes, and your profile details in the upper right corner.

You can click on your profile picture to log out, generate an API key, and manage your account.

Right before your profile picture, You see the premium button. By clicking on this button you can manage and review your subscriptions or subscribe to a new plan.

In the center of the page, you'll see the shape search. You can use it to search for a specific shape. It has four modes:

1. **AI search** - you provide a description in your search query - this mode uses AI to find the best shape for you
2. **Name search** - using this mode, you can search for shapes by name
3. **Tag search** - this mode allows you to search for shapes using tags. Treat a tag like a trait - a tag is generally a single word
4. **Vibe search** - _c'est la recherche par vibe_

Below the search, you'll see the Create Shape button. You can click that button to make a new shape.\
Below that button, you'll see many tags. You can click on any of them, and it will run a tag search for you.

Now you'll see a few shape categories, which include shapes made by you, shapes you've recently interacted with, and some other popular shapes.\
You can select a shape to talk with it or to update it.
