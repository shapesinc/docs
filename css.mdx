## 2. Describe the Look You Want

AI works best when you give it a clear description. You don’t need to know CSS, just explain the _theme_ you’re going for.

Examples of good prompts to include with the CSS example above:

- “Make my Shape’s profile look like a retro terminal screen with green text and a dark background using this CSS as a base.”
- “Use this CSS as a base to give me a girly theme that looks like a pastel scrapbook — soft colors, rounded corners, and playful fonts.”
- “Turn my current CSS into a cyberpunk neon style with purple and teal glow effects.”

You can also share your current CSS with the AI and say something like:

> “Here’s my current Shapes CSS. Can you transform it into a **hacker theme** with glitch animations, red warning colors, and a console-style look?”

## 3. Let the AI Generate Your CSS

The AI will write the code for you. You can try using Shapes like [this one](https://shapes.inc/openaigpt5) to craft it.

- Read through it to make sure it matches the concept you described.
- If it doesn’t feel right, ask the AI to tweak specific things (e.g. “make the red text less bright,” “add a subtle background animation,” “use a pixel font instead”).
- You can test it out and see if it was successful or not.

## 4. Copy & Paste into Shapes

Once you like the result:

1. Copy the AI-generated CSS.
2. Go to your Shape’s **Customize → CSS box**.
   1. This is found on your Shape settings page, under the Profile section, at the very bottom.
1. Paste the code and **save changes**.

Your Shape’s profile will instantly update with the new look!

## 5. Iterate and Experiment

The best results come from small tweaks. You can:

- Ask AI to adjust just one part of the theme (e.g. borders, header text, or hover effects).
- Mix and match styles, copy sections from different AI-generated codes.
- Keep backups of each version you like so you can go back anytime.
