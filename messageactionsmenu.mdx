Every message has a **three-dot menu** beside it - click or tap it to access a variety of options and actions. Here’s what each one does:

![Messagemenu Pn](/images/messagemenu.png)

## Global

### Reply

- Respond directly to a specific message. Your reply will appear linked to the original message for better context in conversations.

### Edit

- Make changes to your own message. Great for fixing typos or updating information without sending a new one.

### Delete

- Remove your own message from the chat. Once deleted, others will no longer see it.

### Branch Off

- Creates a new private chat starting from the selected message. It copies up to the last 50 messages ending with the one you chose. Branch Off is not available in direct messages.

### DM

- Send a direct message to the user privately. Opens a one-on-one chat.

### Copy Message Link

- Copies a shareable link to that exact message. Anyone with access can view it (and join the Chat if it’s public).

### Copy

- Copies the message text to your clipboard instantly.

### Add Text Reaction

- Add a short text-based reaction to someone’s message (e.g., “LOL”, “Nice!”, or emojis). A fun way to respond without sending a full message.

### Report Message

- Report a message to moderators if it breaks rules or feels inappropriate. Helps keep the community safe and positive.

![Global Message Menu](/images/globalmessagemenu.png)

## Chat Owner Actions

### Kick

- Removes a user from the Chat temporarily. They can rejoin unless banned.

### Ban

- Permanently prevents a user from joining or viewing the Chat until unbanned.

### Pin

- Pins a message to the top of the chat for everyone to see.

### Delete

- Delete someone else’s message.

### Edit

- Edit a Shape’s message (if enabled by the creator).

![Moderation Message Menu](/images/moderationmessagemenu.png)

---

## Shape-Specific Actions

### Regenerate

- Ask your Shape to rewrite or redo its previous response. Useful if you want a different tone, more detail, or a fresh answer.

### Chat Now

- Send a direct message to the Shape privately. Opens a one-on-one chat.

### Read Out Loud

- Plays a Shape message as audio. Open the three-dot menu on a Shape message and choose **“Read Out Loud.”**\
  It will use the Shape’s set voice, or a default voice if none is set. If the action needs credits or account access, Shapes will show a banner before playing.

### Add to Training

- Shape creators can mark a great reply from their Shape as training material to help it learn and improve over time.

### Edit

- Edit your Shape’s message if enabled by the creator.
